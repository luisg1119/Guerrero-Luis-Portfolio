You can view a video of the Collaborative Editing tool at: https://www.youtube.com/watch?v=_4ASffAV2oI&feature=youtu.beQuick Set-Up:To use the Collaborative Editing tool open one of the jar files and enter the necessary information, for host you can either enter “localhost” for local computers, or an IP address if you are wanting to connect remotely. In order to connect to another Collaborative Editing window open the second jar file with the same port and host information as the first. Note: Please disable any firewalls if you are trying to use an IP address as a host address, otherwise the collaborative tool will not connect to the host IP.
